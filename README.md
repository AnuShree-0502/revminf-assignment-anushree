# RevMind AI Chatbot

AI Chatbot built with Next.js 14 + Groq API for RevMind Labs Fresher Assignment 2026.

## Features
- Real-time streaming chat with Groq Llama 3.3 70B
- Built with Next.js 14 App Router + TypeScript
- Deployed on Vercel

## Run Locally
```bash
npm install
echo "GROQ_API_KEY=your_key" > .env.local
npm run dev
